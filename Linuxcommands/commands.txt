Linux commands
Ctrl + alt + T - команда открытия терминала
~ - символ домашней директории
/ - символ корневой директории 

. - обозначает текущую директорию
.. - обозначает предыдущую директорию

  1. ls - команда для вывода всех директорий(папка) и файлов в текущей (рабочей) директории в терминале 
  
  2. Cd (change directory) -  команда для перехода из одной директории в другую
  
  3. mkdir - создает директории
  
  4. touch - создает файл
    
  5. rmdir - удаляет пустую директорию
  
  6. rm - удаляет файл
  
  7. rm -rf - удаляет вообще всё (принудительное удаление)
  
  8. ls -a - показывает скрытые и обычные файлы и директории 
     ls -la - показывает подробную информацию
     
  9. pwd(print working directory) - выводит путь до текущий директории 
  
  10.  cp - копирует файл или директорию в указанное место
             cp <название>   <куда>
  
  11. mv -  команда для перемещение файла или директории, а также для 
  переименовывания файла или директории
             переименовывание: mv john.txt jamie.txt 
             перемещение: mv john.txt directory/
  
  12. cat - показывает содержание файла
  
  13. nano - открывает файл для редактирование в графическом интерфейсе Нано 
  *Если нет файла, то нано создаст новый с таким же иминем
  
  14. echo -  команда для быстрой записи текста в файл 
          echo текст > файл
